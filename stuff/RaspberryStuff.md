# The Raspberry&nbsp;Pi

Common information about the Raspberry&nbsp;Pi, my projects based on that computer, and other useful stuff around the "Pi". Refer also to the [page with the Linux Stuff](LinuxStuff.md) for information related to the Linux operating system in general.

   1. [General Documentation](https://www.raspberrypi.com/documentation/computers/raspberry-pi.html)\
      The official documentation for Raspberry Pi computers and microcontrollers. The Raspi Imager can be found [here](https://www.raspberrypi.com/software/).
   
   2. [The Raspi Camera](raspi/CameraStuff.md)\
      Information on how to install the Raspberry Camera, and how to use it.
      
   3. [The Raspi as a USB Device](raspi/RaspiAsKeyboard.md)\
      Information on how to use a Raspberry&nbsp;Pi as a USB device (like a keyboard or a harddrive) for another computer. Mainly Raspberry&nbsp;Pi&nbsp;Zero models are used for this purpose, but also Pi&nbsp;4 and Pi&nbsp;5 can be used this way.
      
   4. [Installing a Compute Module](raspi/InstallCM.md)\
      How to install the operating system on a CM5.
      
   5. [Switching off the Undervoltage Warning](raspi/Undervoltage.md)\
      The Raspberry&nbsp;Pi&nbsp;5 is quite particular about power supplies, with the result that a warning appears when a not fully compliant power supply is used.

   6. [Getting rid of `/dev/zram0`](raspi/ZRAM.md)\
      When installing the Raspberry&nbsp;Pi&nbsp;OS it uses a device `/dev/zram0` for swapping into memory.
      
   7. [Setting up a Music Server](MusicServer.md)\
      How to setup a music server based on [OwnTone](https://owntone.github.io/owntone-server/). This software does not only run on a Raspberry&nbsp;Pi, but also on various other Linux versions.
      
   8. [PiJuice](raspi/PiJuice.md)\
      The PiJuice hat provides a UPS to the Raspberry Pi.
      
   9. [The Remote Desktop Protocol](raspi/RDP.md)\
      How to use the RDP with a Raspberry&nbsp;Pi.
